## 📊 Project Overview
This project presents an interactive **Car Sales Analytics Dashboard** built using Power BI.  
It provides insights into vehicle sales performance, customer trends, and market behavior through dynamic visualizations.

The dashboard is designed with a **modern dark UI**, inspired by real-world business intelligence systems.

---

## 🎯 Key Features

- 📈 Sales Performance Analysis
- 🚗 Car Model & Brand Insights
- ⚙️ Transmission & Fuel Type Analysis
- 🎨 Color Preference Trends
- 🌍 Market & Sales Distribution
- 🖱️ Interactive Navigation (Page Buttons & Icons)
- 🧠 Tooltip-based Detailed Insights

---

## 📑 Dashboard Pages

### 🏠 1. Home
- Navigation menu with icons
- Entry point for dashboard

---

### 📊 2. Car Sales Analytics Overview
- Total Sales, Total Cars, Average Price, Mileage
- Sales by Year trend
- Fuel Type distribution
- Color-based analysis
- Dynamic car image based on selection

---

### 🚗 3. Brand & Model Analysis
- Transmission analysis (Automatic vs Manual)
- Year-based trends
- Car feature insights (e.g., Backup Camera)
- Interactive slicers for model filtering

---

### 🌍 4. Sales & Market Insights
- Total Orders & Sales overview
- Sales by Condition (New / Used)
- Transmission-based sales split
- Color-based demand trends
- Year-wise sales pattern

---

## 🧮 Tools & Technologies

- 📊 Power BI Desktop
- 📁 DAX (Data Analysis Expressions)
- 🎨 Custom Theme JSON
- 🖼️ GitHub Image Hosting (Dynamic Image URLs)

---

## 🧠 Key Insights

- Sales trends show variation across years with peak periods
- Automatic transmission dominates in total sales
- Certain colors (e.g., Red, Gray) are more popular
- Fuel type distribution indicates balanced market demand
- Model-level analysis helps identify top-performing vehicles

- Synthetic Car Sales Dataset
- Includes:
  - Car Make & Model
  - Year
  - Price
  - Mileage
  - Fuel Type
  - Transmission
  - Condition
  - Color

## 🎨 Design Highlights

- Dark theme (#121212) for professional UI
- Consistent color palette (Blue, Red, White)
- Rounded cards & modern layout
- Icon-based navigation (like web apps)

---

## 🚀 How to Use

1. Open `.pbix` file in Power BI Desktop
2. Use slicers to filter data
3. Navigate pages using icon buttons
4. Hover over visuals for tooltip insights
